Connect to WhatsApp

How to connect:

  1. Enter your WhatsApp number with country code no "+" No bracket no space
  2. Click "Request Pairing Code"
  3. Copy the 8-digit code
  4. Open WhatsApp → Settings → Linked Devices → Link a Device
  5. Paste the code when prompted
